# 序章

## 技术支持
- JavaScript高级程序设计
- 深入浅出Node.js
- ECMAScript 6 入门

## 环境准备
- node: v8.6.0
- MongoDB: 3.4.9
- 部署到阿里云的环境
    - CPU：1核 内存：1GB
    - CentOS 7.4 64位

## 框架
```sh
├── config.js             # 配置文件
├── public                # 静态资源目录
│   ├── blogs             # 博客文章目录
│   ├── images            # 图片资源目录
│   ├── scripts           # 静态脚本目录
│   └── styles            # 样式目录
└── server                # 后端代码目录
    ├── db                # 数据库目录
    ├── middlewares       # 自定义中间件目录
    ├── models            # 数据模型目录
    ├── routers           # 路由目录
    ├── services          # 逻辑服务目录
    ├── utils             # 工具目录
    ├── views             # 视图目录
    └── app.js            # 入口文件
```
